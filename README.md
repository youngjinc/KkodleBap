## 🚀 Features
- 총 6번의 기회로 자모 6개를 조합해 정답 맞추기
- 게임 결과를 복사해 공유하기
- 유저 간 점수를 비교하여 랭킹을 제공(차기 릴리스 목표)
- 하루 세 번 힌트 제공(차기 릴리스 목표)

<br>
